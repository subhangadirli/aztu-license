# AzTU License v2.1

### Strong Open Continuity License

---

## 🌍 Overview

**AzTU License v2.1** is a strong copyleft open source license designed to ensure one core principle:

> **Software must never become closed source.**

It is inspired by the philosophy of the GNU Affero General Public License (AGPL), with a clearer and more explicit focus on **perpetual openness**.

---

## 🎯 Goals

This license exists to:

* Guarantee that software remains open source forever
* Prevent proprietary forks
* Ensure source code availability in all use cases (including SaaS)
* Provide a simpler and more readable alternative to complex legal texts

---

## ⚖️ Key Features

### 🔓 Perpetual Openness

All derivative works must remain open source.
Closing the source code is strictly prohibited.

---

### 🔁 Strong Copyleft

If you modify or distribute this software:

* You must release your changes under the same license or a compatible one
* You must provide full access to the source code

---

### 🌐 Network Use Protection (SaaS)

If the software is used over a network:

* Users must be able to access the full corresponding source code
* This includes APIs, cloud services, and hosted applications

---

### 🚫 No Additional Restrictions

You may not:

* Add DRM (Digital Rights Management)
* Restrict user freedoms through contracts
* Hide or withhold source code

---

### 🧾 Attribution Required

You must retain:

* Copyright notices
* Author names
* This license text

## 🤝 Relationship with AGPL

This license is:

* Inspired by the principles of the GNU Affero General Public License
* Designed to achieve similar goals in a more explicit and simplified way

In practice:

* It enforces strong copyleft
* It closes the SaaS loophole
* It guarantees continuous openness

However:

* It is **not an official replacement** for AGPL
* It is a separate license with its own structure

---

## 🧠 Why Not Just Use AGPL?

AGPL is powerful, but:

* It is long and complex
* Many developers find it difficult to understand
* Its intent is not always explicit

AzTU License aims to:

* Make the philosophy clearer
* Emphasize “never closed source” as a core rule
* Improve readability without losing strength
